Armani Os

ArmaniOS is an Arch-based distribution designed for both low-end PCs and high-end PCs. There is a package of necessary utilities (yay, firefox, etc.), a set of various desktop environments (KDE Plasma, Gnome, XFCE, UKUI, Cutefish, TDE or just Trinity). The distribution is oriented both for beginners and professionals.

Our main Desktop Environment

KDE Plasma

KDE Plasma, a graphical desktop environment with customizable layouts and panels, supporting virtual desktops and widgets. Written with Qt 5 and KDE Frameworks 5.

XFCE

Xfce or XFCE (pronounced as four individual letters) is a free and open-source desktop environment for Linux and other Unix-like operating systems.Xfce aims to be fast and lightweight while still being visually appealing and easy to use.

GNOME

GNOME is a free desktop environment for UNIX-like operating systems. GNOME is not part of the GNU project.

ArDE

ArmaniOS Desktop Environment with XFWM4 and PolyBar

F*ck SystemD

We use SystemD, but if you want OpenRC - #FUCKSYSTEMD is here.OpenRC is a dependency-based init system for Unix-like computer operating systems. It was created by Roy Marples, a NetBSD developer who was also active in the Gentoo project. It became more broadly adopted as an init system outside of Gentoo following the decision by some Linux distributions not to adopt systemd.

How to migrate OpenRC?

1. Check if you have internet connection
2. Open Terminal
3. Write fucksystemd and press enter
